Professional academic website for Abdelrahman O. Sawy.

## Project Structure

```
AOSAWY.ORG Website Project/
├── docs/

│   ├── mockups/         ← HTML mockups from brainstorming session
│   └── assets/          ← Reference assets (screenshots, inspiration)

├── content/             ← Pre-staged content (YAML/MD) before moving to src/
└── src/                 ← Astro project lives here (created during build)
```
