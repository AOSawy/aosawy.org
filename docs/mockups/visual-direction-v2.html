Design Language Applied

How Apple + Liquid Glass principles shape every element

🍎 Apple Design Language

  • SF-inspired Inter font (closest web equivalent)
  • Pill-shaped buttons (border-radius: 980px)
  • Generous whitespace & breathing room
  • Blue-#0071e3 accent (Apple's exact link blue)
  • Hairline 0.5px borders throughout
  • Subtle hover lift on interactive cards

✨ iOS 26 Liquid Glass

  • Frosted glass nav: backdrop-filter blur(20px)
  • Translucent card backgrounds (rgba white 0.65)
  • Saturate(180%) for that glass material depth
  • Ultra-thin borders with near-zero opacity
  • Soft shadows (0 1px 4px) — no hard edges
  • #f5f5f7 background (Apple's signature gray)

📚 Academic Authority

  • Playfair Display serif for headings (name, page titles)
  • Inter sans-serif for body (readability, professionalism)
  • Scholarly section labels (uppercase, tracked out)
  • Publications with PDF/Code/BibTeX links
  • Research group affiliations prominent

⚡ Technical Details

  • Tailwind CSS for utility classes (Apple-like precision)
  • CSS custom properties for theming
  • Scroll-triggered animations (Intersection Observer)
  • System font stack fallback for performance
  • Responsive: mobile-first, glass adapts to viewport
